# quiz

Pequeno site para gerar quiz com ajuda de llm

## Como funciona

- Use a tela inicial para gerar um prompt informando os assuntos, dificuldade e quantidade de questões.
- Responda as questões.
- Receba um prompt informando erros e acertos para ter retorno do que reforçar no estudo.
